# Snake Game

A simple snake game written in C for the Windows console.

## Controls

- `W` move up
- `S` move down
- `A` move left
- `D` move right
- `Q` quit

## Build

If you have MinGW GCC installed, run this inside the project folder:

```powershell
gcc main.c -o snake.exe
```

Then start the game with:

```powershell
.\snake.exe
```

## Symbols

- `@` snake head
- `O` snake body
- `*` food
- `#` wall
